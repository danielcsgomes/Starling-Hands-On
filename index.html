Starling Hands On by danielcsgomes

These are my experiments with Starling Framework and Stage3D.

To be able to test them you need to do three things:

  1. Create a new AS3 Project.
  2. Add to your build path the required libs including this one.
  3. Add the wmode = "direct" to your html template.

Required Libs

  • Starling Framework
  • Starling Extension: Particle System
  • Stats

Add them to your Build Path.

Source Files

  • SquareRotation source files
  • Touch and Follow Fire source files
